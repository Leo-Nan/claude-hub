# Agent 画布设计文档

**日期**: 2026-02-16

## 概述

将现有的底部 Agent 面板改造为右侧独立的 Agent 配置管理画布，提供 Agent 配置管理和快速跳转功能。

## 目标

- Agent 配置管理中心（规则、指令、系统提示词）
- 点击 Agent 卡片跳转到终端对应会话

## UI 设计

### 布局结构

```
┌─────────────────────────────────────────────────────────────┐
│  主界面 (终端)                                              │
│                              │  Agent 画布 (右侧面板)        │
│                              │  ┌─────────────────────────┐ │
│                              │  │ Agent 列表  │ 配置编辑   │ │
│                              │  │            │            │ │
│                              │  │ [卡片 1]   │ 规则/指令  │ │
│                              │  │ [卡片 2]   │ 系统提示词 │ │
│                              │  │ [卡片 3]   │            │ │
│                              │  └─────────────────────────┘ │
└─────────────────────────────────────────────────────────────┘
```

- **左侧区域**：Agent 卡片列表
- **右侧区域**：配置编辑区
- **面板宽度**：可调整，默认 400px
- **展开/收起**：点击按钮切换显示

### Agent 卡片设计

```
┌────────────────────────────┐
│ ● Agent 名称          [活跃]│
│   能力: skill1, skill2     │
│   [启动] [配置] [删除]     │
└────────────────────────────┘
```

- 状态指示点（活跃/空闲/思考中）
- Agent 名称
- 能力标签
- 快捷操作按钮
- 点击卡片 → 右侧显示配置 + 跳转到终端会话

### 配置编辑区

支持编辑：
1. **规则文档** - Agent 行为规范
2. **系统提示词** - System Prompt
3. **能力列表** - 可用技能

## 功能列表

1. **右侧面板容器** - 可展开/收起
2. **Agent 卡片列表** - 显示状态、名称、能力
3. **配置编辑区** - 规则、指令、提示词编辑
4. **点击跳转** - 卡片点击跳转到终端对应会话
5. **Agent CRUD** - 添加、编辑、删除 Agent

## 数据结构

```typescript
interface Agent {
  id: string;
  name: string;
  status: 'active' | 'idle' | 'thinking';
  skills: string[];
  rules: string;        // 规则文档
  systemPrompt: string; // 系统提示词
  projectId: string;
}
```
